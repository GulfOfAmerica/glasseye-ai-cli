# GitHub Repository Setup Guide

## Quick Setup

1. **Create GitHub Organization/Account**
   - Organization: `glasseye-ai`
   - Or use personal account

2. **Create Repository**
   ```bash
   # On GitHub.com
   Name: glasseye-cli
   Description: Security Intelligence CLI - Context-aware vulnerability scanning
   Public: ✓
   ```

3. **Push Code**
   ```bash
   cd /home/x/glasseye-cli
   git init
   git add .
   git commit -m "Initial commit: Glasseye AI CLI v1.0.0"
   git branch -M main
   git remote add origin git@github.com:glasseye-ai/glasseye-cli.git
   git push -u origin main
   ```

4. **Add Topics on GitHub**
   - security
   - vulnerability-scanner
   - ai
   - npm
   - cli
   - cybersecurity
   - devsecops

5. **Enable GitHub Pages (for website)**
   ```bash
   cd /home/x/glasseye-website
   git init
   git add index.html
   git commit -m "Add landing page"
   git branch -M main
   git remote add origin git@github.com:glasseye-ai/glasseye-ai.github.io.git
   git push -u origin main
   ```
   
   Then on GitHub: Settings → Pages → Source: main branch
   Website will be live at: https://glasseye-ai.github.io

## npm Publishing

```bash
cd /home/x/glasseye-cli
npm login
# Enter credentials
npm publish --access public
```
