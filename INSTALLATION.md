# Installation

## Quick Install (GitHub)

Install directly from GitHub:

```bash
npm install -g https://github.com/GulfOfAmerica/glasseye-ai-cli.git
```

## Alternative: npm Registry

Once published to npm:

```bash
npm install -g @glasseye/cli
```

(Requires npm account to publish first)

## Verify

```bash
glasseye --version
glasseye scan
```
